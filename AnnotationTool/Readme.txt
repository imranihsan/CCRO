CSAT- Citation Sentiment Analysis Tool
----------------------------------------------------

Installation Steps:

	
	1. 	Install Visual Studio.The version of the visual studio should above than 2012

	2.	Save the folders your hard disk.
        3.	Make new Project in the visual studio and than browse to the folders copied earlier.

	4.	Use NuGet Manager (Or other methods available) to install the libraries provided in the dependancies folder.

	5.	Create respective forms, using the code provided in the Backend and Front end folders.

	6. 	Run the application.It will going to take 1 min to process.
	
	7.	Your application will now run.
